# ☕ Exercícios 1 - Fundamentos da Linguagem de Programação Java (Bootcamp Santander)

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do **Bootcamp Santander**, com o objetivo de praticar os fundamentos da linguagem Java por meio da resolução de exercícios envolvendo entrada de dados, operações matemáticas e lógica de programação.

Além da implementação dos exercícios propostos, a aplicação foi estruturada utilizando um **menu interativo**, permitindo ao usuário selecionar qual exercício deseja executar sem precisar reiniciar o programa.

O sistema permanece em execução até que o usuário escolha a opção de encerramento, proporcionando uma experiência semelhante à de aplicações baseadas em menus.

---

## ⚙️ Funcionalidades

O menu principal oferece acesso aos seguintes exercícios:

### 1️⃣ Cálculo de Idade

Recebe o nome e o ano de nascimento do usuário e calcula sua idade.

### 2️⃣ Área do Quadrado

Recebe o valor do lado do quadrado e calcula sua área.

**Fórmula utilizada:**

```text
Área = lado × lado
```

### 3️⃣ Área do Retângulo

Recebe os valores da base e da altura e calcula sua área.

**Fórmula utilizada:**

```text
Área = base × altura
```

### 4️⃣ Diferença de Idade

Recebe o nome e a idade de duas pessoas e calcula a diferença de idade entre elas.

### 0️⃣ Sair

Encerra a aplicação.

---

## 🏗️ Estrutura da Aplicação

O projeto foi organizado utilizando:

- Classe principal com método `main`
- Menu interativo utilizando `while`
- Estrutura de decisão `switch case`
- Métodos separados para cada exercício
- Entrada de dados utilizando `Scanner`
- Organização do código visando reutilização e manutenção

## 🎯 Exercícios Desenvolvidos

### 1️⃣ Cálculo de Idade

Desenvolver um programa que receba:

- Nome
- Ano de nascimento

E exiba a mensagem:

```text
Olá Fulano, você tem X anos.
```

---

### 2️⃣ Área do Quadrado

Desenvolver um programa que receba o tamanho do lado de um quadrado e calcule sua área utilizando a fórmula:

```text
Área = lado × lado
```

---

### 3️⃣ Área do Retângulo

Desenvolver um programa que receba:

- Base
- Altura

E calcule sua área utilizando a fórmula:

```text
Área = base × altura
```

---

### 4️⃣ Diferença de Idade

Desenvolver um programa que receba:

- Nome da primeira pessoa
- Idade da primeira pessoa
- Nome da segunda pessoa
- Idade da segunda pessoa

E exiba a diferença de idade entre elas.
